# Interconnection Decision Tool

This tool provides a simplified, non-proprietary framework for evaluating renewable energy project advancement decisions at key interconnection milestones.

## Features
- Risk-adjusted NPV modeling
- ITC probability & safe harbor logic
- Contracted vs merchant revenue separation
- Basis and congestion sensitivity
- Solar, BESS, and hybrid configurations
- Levered-lite financing view

## Purpose
Designed to replicate investment committee-style decision logic without requiring proprietary internal data.
